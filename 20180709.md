#### [문제4] 리스트의 append와 extend의 차이

```
>>> a = [1, 2, 3]
>>> a.append([4,5])
>>> print(a)

[1, 2, 3, [4, 5]]
```

```
>>> a = [1, 2, 3]
>>> a.extend([4,5])
>>> print(a)

[1, 2, 3, 4, 5]
```



#### [문제5] 리스트 정렬

```
>>> a = [1, 3, 5, 4, 2]
>>> a.sort()
>>> a.reverse()
>>> print(a)

[5, 4, 3, 2, 1]
```

sort()는 값을 정렬해주고, reverse()는 값을 반대 순서로 뒤집는다.
